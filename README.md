# Penta-B

---

A React and vanilla JavaScript simulation of  rover on Mars.

## life url

- https://penta-b.vercel.app/

## Available Scripts

- to initialize the app

```shell
    yarn init
```

- to build the app

```shell
    yarn run build
```

- to start development server

```shell
    yarn run dev
```

- to test

```shell
    yarn run test
```
